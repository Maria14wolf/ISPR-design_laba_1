# ISPR-design_laba_1
Лабораторна робота №1 з предмену Проєктування ІСППР
Кнопки “Analyze by keyword: Computer” та “Analyze by keyword: SQL” запускають функцію семантичного аналізу й перебудовують дерево WordTree відповідно до вибраного ключового слова. Після натискання кнопки схема відображає всі фрази, у яких зустрічається це слово, та показує контекст його використання в тексті. Перша схема (Semantic Network) демонструє ієрархічну структуру навчальних курсів і предметів, тобто зв’язки типу курс → дисципліна → тема. Друга схема (Semantic Analysis) показує семантичні зв’язки слова з іншими поняттями, дозволяючи виявити, у яких змістових контекстах використовуються Computer або SQL.
